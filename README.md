🤖 CSS AI Generator

Uma aplicação web que utiliza Inteligência Artificial para gerar códigos CSS a partir de descrições em linguagem natural.

O usuário pode digitar um prompt como "bola azul pingando", e a aplicação gera automaticamente o código CSS correspondente, exibindo o resultado visual em tempo real em uma área de preview ao lado.

O projeto tem foco em experimentos com IA aplicada ao front-end, facilitando a criação de estilos e animações CSS.
